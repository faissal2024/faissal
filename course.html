Your way to learn website programming

Between three weeks and two months

Learn HTML Advanced

After learning the basics, in a week or two you will need to learn advanced topics such as: Forms: How to create forms to collect data. Embedding: Such as embedding video and audio. Advanced elements such as tables and dropdowns. Interacting with Script to improve the user experience.

Learn CSS Basics in 1-2 Weeks

CSS basics include

Formatting text, colors, and backgrounds. Controlling margins, padding, and borders. Formatting dimensions (such as width and height). Aligning elements using properties such as text-align, vertical-align, and display. Positioning such as relative, absolute, fixed, and sticky

Learn Advanced CSS Techniques in 4-5 Weeks

After learning the basics, you can move on to advanced topics such as:

Flexbox: To create flexible and connected designs. CSS Grid: To create complex layouts using a grid of rows and columns. Responsive Design: Using media queries to make pages work well on different devices (mobile, tablet, computer). Animations (CSS Animations) and effects such as fading or moving. CSS Frameworks such as Bootstrap and Tailwind CSS

Learn the basics of Script in 3 to 5 weeks

Script basics include:

Variables (let, const, var). Basic data types (such as numbers, strings, booleans). Arithmetic and logical operations (addition, subtraction, comparisons). Loops (for, while). Conditions (if, else, switch). Arrays and objects. Functions.

Learn advanced Script concepts in 5-7 weeks

After you master the basics, you should learn advanced concepts such as:

DOM (Document Object Model): Manipulate HTML elements via Script. Event Handling: Interacting with clicks, animations, and keys. Advanced Arrays and Objects: Learn advanced techniques for handling arrays and objects. Asynchronous Programming: Learn Promises and async/await to handle asynchronous operations. Anonymous Functions and Callbacks. Functional Programming: Learn functional programming methods such as transactions and higher-order functions..

Finally building real projects

Portfolio Websiokokte

Skills used: HTML, CSS, Script. Project Description: Create a website that contains your resume, personal information, links to previous projects, and contact details. Project Objective: Apply the basics such as structure, formatting, adding images, and links. You can use CSS to make the site attractive and Script to add interactivity such as animation effects.
